# SVNC 用户手册 (User Manual)

**版本**: V1.0  
**日期**: 2025-12-30

---

## 1. 简介
SVNC 是一款现代化的 SVN 图形客户端，专注于提供流畅、美观且高效的版本控制体验。它简化了复杂的 SVN 指令，让您通过简单的拖拽和点击即可完成文件管理。

---

## 2. 快速开始

### 登录
1. 打开应用。
2. 输入 **Repository URL** (如 `https://svn.example.com/repo`).
3. 输入 **Username** 和 **Password**。
4. 点击 "Login"。

### 界面概览
- **顶部工具栏**: 
  - `←`: 返回上一级。
  - `地址栏`: 显示当前路径，支持手动输入跳转。
  - `刷新`: 重新加载当前目录。
  - `新建文件夹`: 在当前目录下创建新目录。
  - `上传`: 点击选择文件上传。
  - `设置 (Settings)`: 自定义文件打开方式。
  - `退出登录`: 返回登录页。
- **文件列表**: 显示文件名、大小、图标。
- **底部状态栏**: 显示选中项数量及操作按钮 (History, Open, Delete)。

## 3. 功能详解

### 3.1 浏览与导航
- **进入文件夹**: 双击文件夹，或点击文件夹左侧的小箭头展开（树状视图）。
- **返回**: 点击顶部 `←` 按钮。
- **复制路径**: 点击列表右侧的 "Copy" 图标。
- **查看属性**: 点击列表右侧的 "Info" (i) 图标。

### 3.2 文件预览与打开
- **应用内预览**: 
  - 默认情况下，双击文本文件 (txt, md, js, json...) 或图片 (png, jpg) 会直接在应用内弹出预览窗口。
- **外部程序打开**:
  - 您可以在设置中配置特定文件（如 `.xlsx`）使用系统默认程序打开。
  - 操作：双击该文件，或选中后点击底部 "Open"。

### 3.3 下载
- **单文件下载**: 点击列表右侧的 "Download" 图标。
- **批量下载**: 选中多个文件（按住 Ctrl/Cmd 或 Shift 选择），点击底部 "Download All"。
- **选择保存位置**: 系统会弹出对话框询问保存路径。

### 3.4 上传与更新
- **上传文件**: 
  - **拖拽**: 直接将文件从电脑桌面拖入应用窗口。
  - **按钮**: 点击顶部 "Upload Cloud" 图标。
- **自定义提交信息**: 上传时会弹出对话框，您可以输入通过 Commit Message (备注)。
- **智能覆盖 (Smart Overwrite)**:
  - 如果上传的文件名已存在，系统会自动执行 `update` -> `commit` 流程。
  - **优势**: 该文件的 SVN 历史记录会被完整保留，而不是被作为新文件替换。

### 3.5 版本历史 (Time Travel)
1. 选中一个文件。
2. 点击底部出现的 **"History"** 按钮。
3. **查看日志**: 弹框显示该文件的所有提交记录（版本号、作者、时间、备注）。
4. **历史预览**: **双击** 历史列表中的任意一行，可以查看该文件在那个历史时刻的内容（支持文本和图片）。

### 3.6 设置 (Settings)
- 点击顶部 "Settings" 图标。
- **文件关联**:
  - 输入扩展名 (如 `docx`)。
  - 选择动作: `Internal Preview` (预览) 或 `External App` (外部打开)。
  - 点击 "Add"。

---

## 4. 注意事项
1. **网络连接**: 请确保您的网络可以访问 SVN 服务器。
2. **SVN 命令行**: 本软件依赖系统底层的 `svn` 命令。如果在 macOS 上遇到问题，请确保已安装 Command Line Tools (`xcode-select --install`).
3. **大文件预览**: 预览过大的文本文件可能会导致界面卡顿，建议对大文件使用"下载"功能。

---

## 5. 常见问题 (FAQ)

**Q: 为什么上传覆盖时提示 Error?**  
A: 请检查您是否有该目录的写权限。如果文件被锁定，解锁后再试。

**Q: 双击 xlsx 文件没有反应？**  
A: 请去 Settings 中添加 `xlsx` 并设为 `External`。如果仍不行，请检查您的电脑是否安装了 Excel 或 WPS。
